# Personal Blog Project

Nextjs 기반 포트폴리오용 개인 블로그 제작

[https://ricksw-dev.vercel.app/](https://ricksw-dev.vercel.app/)

## Tech Stack

### Frontend ( Only )

- Nextjs
- Typescript
- emotion (css)

### Deploy

- Vercel

## Custom Directory

### components/

페이지 내에서 사용하는 컴포넌트들 (기능별)

### constants/

간단한 공통 데이터 관리

### container/

실제 페이지를 이루는 레이아웃 및 컨테이너

### lib/

공통 Util 파일 관리

### posts/

블로그 내 포스팅을 markdown 파일로 저장하여 관리

### types/

공통 Type 관리
